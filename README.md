# POLARIS Registry

Repositorio inicial para desarrollar el **portal POLARIS Registry**, una plataforma de membresía digital del programa **POLARIS**.

POLARIS (Programa de Oportunidades para el Liderazgo y el Aprendizaje en la Revolución de la Industria eSpacial) es un programa universitario enfocado en la formación de estudiantes en tecnologías espaciales del **Instituto Tecnológico de Matamoros**.

Este repositorio contiene la **estructura base del portal**, la cual será ampliada y mejorada durante el desarrollo del proyecto en el curso.

---

## Objetivo del proyecto

Construir un portal web que permita:

- Presentar el programa POLARIS
- Mostrar los equipos y proyectos del programa
- Mostrar eventos y actividades
- Permitir que estudiantes se registren como miembros del programa

El sistema final funcionará como una **plataforma de registro y membresía digital para participantes del programa POLARIS**.

## Primera actividad

Construir la primera versión del **portal principal (`index.html`)**.

Debe incluir al menos:

- Título del portal
- Breve descripción del programa
- Sección de proyectos
- Botón **"Únete a POLARIS"**

El botón debe dirigir a:
registro.html


---

## Desarrollo del portal

El desarrollo del portal deberá basarse en el documento proporcionado en clase:

**Portal del Programa POLARIS (PDF)**

A partir de ese documento deberán estructurar el sitio agregando secciones como:

- Inicio
- ¿Qué es POLARIS?
- Modelo de formación
- Equipos del programa
- Eventos
- Desarrollo tecnológico
- Investigación
- Galería
- Noticias
- Registro de participantes

## Reglas del proyecto

- Todo el trabajo debe subirse al repositorio.

- El desarrollo del sistema será progresivo durante el curso.

Se evaluará:

- estructura del portal

- formularios

### Misión 4 – Herramientas Frontend y diseño
- Mejor diseño
- Mejor UX

### Misión 5 – Pruebas de interfaces

Los equipos acumularán puntos durante cada misión.
